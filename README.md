# 🎨 Modern Web Image Editor

A sleek, lightweight, and modern image editing application built entirely with vanilla web technologies. It features a premium dark-mode UI and leverages the HTML5 Canvas API to apply real-time filters and adjustments to your photos.

## ✨ Features

* **Sleek Dark UI**: A modern, eye-friendly interface built with CSS variables and Flexbox.
* **Image Upload**: Seamlessly load images from your local device onto the editing workspace.
* **Real-time Adjustments**: Custom range sliders to tweak your image instantly:
  * Brightness
  * Saturation
  * Contrast
  * Blur
* **Quick Presets**: One-click filters (e.g., Grayscale, Sepia, Invert, Vintage) to speed up your workflow.
* **Export & Save**: Download your edited masterpiece directly to your device.
* **Fully Responsive**: Adapts perfectly to your browser window.

## 🛠️ Technologies Used

* **HTML5**: Semantic structure and Canvas element for image manipulation.
* **CSS3**: Custom properties (variables), Flexbox layout, hover states, and smooth transitions. No external CSS frameworks used!
* **Vanilla JavaScript**: DOM manipulation, event handling, and the HTML5 Canvas API for pixel-level image processing.
* **FontAwesome**: For lightweight, scalable UI icons.
